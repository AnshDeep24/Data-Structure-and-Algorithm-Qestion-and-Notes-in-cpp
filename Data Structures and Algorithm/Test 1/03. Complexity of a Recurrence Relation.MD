
# Complexity of a Recurrence Relation


#### Which one of the following correctly determines the solution of the recurrence relation with T(1) = 1?

```
T(n) = 2T(n/2) + Logn

```

### Options

O(N)

O(NlogN)

O(N^2)

O(log N)

### SOLUTION:

#### O(N)
